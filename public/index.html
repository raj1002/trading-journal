Setup steps (2 minutes):
1. Go to notion.so/profile/integrations → New integration → name it "Trading Dashboard"
2. Copy the "Internal Integration Secret" (starts with ntn_)
3. Open your Trade Log database in Notion → click ... menu → Connections → add your integration
4. Copy the database ID from the URL: notion.so/THIS-PART?v=...

CORS note: Direct browser calls to the Notion API get blocked by CORS in production. For local testing, use a browser extension like "CORS Unblock" or "Allow CORS". For production, deploy the 15-line Vercel proxy (included in the help below) and select "Via Vercel proxy".

Vercel proxy (api/notion.js) — deploy this one file:
export default async function handler(req, res) { res.setHeader('Access-Control-Allow-Origin', '*'); res.setHeader('Access-Control-Allow-Methods', 'POST, OPTIONS'); res.setHeader('Access-Control-Allow-Headers', 'Content-Type'); if (req.method === 'OPTIONS') return res.status(200).end(); const { token, dbId, cursor } = req.body; const body = { page_size: 100, sorts: [{ property: "Date", direction: "ascending" }] }; if (cursor) body.start_cursor = cursor; const r = await fetch('https://api.notion.com/v1/databases/'+dbId+'/query', { method: 'POST', headers: { 'Authorization': 'Bearer '+token, 'Notion-Version': '2022-06-28', 'Content-Type': 'application/json' }, body: JSON.stringify(body) }); const data = await r.json(); res.status(r.status).json(data); }
